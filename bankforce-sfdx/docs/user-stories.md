# BankForce CRM – User Stories

## Customer Service

### US-001 – Customer Support Request
As a Bank Customer  
I want to request support related to my bank product  
So that I can solve my issue efficiently  

---

### US-002 – Agent Case Handling
As a Customer Service Agent  
I want to receive cases routed by product skill  
So that I can handle requests more efficiently  

---

### US-003 – Business Hours Validation
As a Service Manager  
I want the system to validate business hours  
So that customers receive accurate service availability information  

---

### US-004 – Human Escalation
As a Bank Customer  
I want my request to be escalated to a human agent  
So that complex issues can be resolved properly
